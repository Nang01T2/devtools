概要： Base64 エンコード・デコードツール（オンライン）

Base64 エンコーダー / デコーダーは、テキストやバイナリデータを Base64 に、また Base64 から元に戻す変換をブラウザー内で直接行います。Base64 はバイナリデータを ASCII テキストとして表現するため、URL、JSON、XML、メール添付ファイル、データ URI に安全に埋め込めます。組み込みの検証機能で瞬時にエンコードまたはデコードでき、サーバーには何も送信されません。

使い方

  1. ソースがプレーンテキストかファイルかを選択します。
  2. エンコードするには、入力欄にテキストを入力・貼り付け（またはファイルをアップロード）すると、Base64 の出力がすぐに表示されます。
  3. デコードするには、Base64 文字列を貼り付けます。デコードされたテキストが表示され、検証されます。
  4. 結果をコピーするか、ファイルを扱う場合はダウンロードします。

例

例入力出力
エンコードHello, World!SGVsbG8sIFdvcmxkIQ==
デコードZm9vYmFyfoobar

よくある質問

Base64 は暗号化ですか？

いいえ。Base64 は暗号化ではなくエンコードです。誰でもデコードできるため、セキュリティや機密性は一切提供しません。秘密情報を保護するためではなく、バイナリデータを安全に転送するために使用してください。

なぜ Base64 の出力は入力より長くなるのですか？

Base64 は入力の 3 バイトごとを 4 つの ASCII 文字として表現するため、出力は元のデータより約 33% 大きくなります。このオーバーヘッドは、テキストとして安全に転送するためのトレードオフです。

データ URI とは何ですか？

データ URI は、Base64 を使ってファイルを URL に直接埋め込むものです（例：data:image/png;base64,iVBORw0K…）。別個の HTTP リクエストなしで小さな画像やフォントをインライン化できます。

デコードはサーバー上で行われますか？

いいえ。エンコードとデコードは完全にブラウザー内で実行されるため、機密性の高いデータでも端末内にとどまります。
