Sobre Gerador de Código de Barras

Gere códigos de barras prontos para retalho e logística em simbologias 1D e 2D comuns, incluindo CODE128, EAN-13, UPC e CODE39, com uma pré-visualização em tempo real à medida que escreve. Pode afinar a largura das barras, a altura, as cores, as margens e o rótulo de texto legível por humanos, e depois exportar o resultado como PNG. Toda a renderização acontece do lado do cliente no seu navegador, pelo que os dados que codifica nunca são carregados.

Como usar

  1. Escolha um formato de código de barras, como CODE128, EAN-13, UPC ou CODE39.
  2. Introduza o valor a codificar, respeitando as regras do formato (por exemplo, o EAN-13 espera 12–13 dígitos).
  3. Ajuste a largura das barras, a altura, as margens, as cores e a posição e o tipo de letra do rótulo de texto.
  4. Verifique a pré-visualização em tempo real para confirmar que o código de barras e o rótulo estão corretos.
  5. Copie o código de barras ou descarregue-o como PNG.

Perguntas frequentes

Quantos dígitos precisa um código de barras EAN-13?

O EAN-13 espera 12–13 dígitos. Se fornecer 12 dígitos, o dígito de verificação final é calculado por si; fornecer todos os 13 permite-lhe indicar o dígito de verificação. Entradas não numéricas ou com comprimento errado serão rejeitadas.

Que formatos de código de barras são suportados?

Simbologias 1D e 2D comuns, incluindo CODE128, EAN-13, UPC e CODE39, entre outras. Cada formato tem o seu próprio conjunto de caracteres e regras de comprimento, pelo que o valor que introduzir deve ser válido para o formato selecionado.

Posso descarregar o código de barras como imagem?

Sim. O código de barras é renderizado no navegador e pode ser descarregado como PNG, ou copiado para a área de transferência para colar em etiquetas, documentos ou ferramentas de design.

Os meus dados são enviados para um servidor para criar o código de barras?

Não. A geração do código de barras corre inteiramente do lado do cliente no seu navegador. O valor que codifica permanece no seu dispositivo e nunca é carregado.
