将 PNG 转换为 PDF

把 PNG 截图、扫描件或图形合并成一个 PDF，用于分享或打印。通过拖动重新排列页面，然后导出一个单一文档 — 完全在你的浏览器中完成，不会上传任何内容。

打开 PNG → PDF 转换器

免费 · 无需注册 · 无需上传 — 100% 在你的浏览器中运行

如何将 PNG 转换为 PDF

  1. 打开转换器并添加你的 PNG 文件 — 拖放、从剪贴板粘贴，或选择整个文件夹。
  2. 拖动缩略图，将页面排列成你想要的顺序。
  3. 点击「图片转 PDF」，即可将它们合并成一个文档。
  4. 你的 PDF 会即时下载 — 一切都在你的浏览器中运行，不会上传任何内容。

关于这些格式

PNG

PNG 是一种支持完整透明度的无损格式，非常适合截图、界面图形、徽标，以及任何带有锐利边缘或文字的图像。代价是体积：以 PNG 存储的照片内容比 JPEG、WebP 或 AVIF 大得多。

PDF

PDF 是通用的文档格式 — 它在每台设备上的显示都完全一致，也是人们在分享、打印、发送邮件和归档时所期待的格式。把图片合并成一个 PDF，能将一堆照片或扫描件变成一个有序、易于发送的单一文档。

常见问题

我的 PNG 文件会被上传到服务器吗？

不会。转换完全在你的浏览器中通过 WebAssembly 编解码器运行 — 你的文件永远不会离开你的设备。首次访问之后，转换器甚至可以离线工作。

我可以一次转换多个文件吗？

可以 — 想添加多少文件（或整个文件夹）都行，一批转换完成后，把所有结果打包成一个 ZIP 下载。没有文件数量或大小限制。
