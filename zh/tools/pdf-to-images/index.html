描述

从PDF文件中逐页提取高质量图片。可选择PNG或JPEG输出，调整分辨率（72–288 DPI）和JPEG质量，批量处理多个PDF，预览页面，单独下载或打包为ZIP。

关于 PDF 转图像转换器

将 PDF 的页面提取为单独的图像文件，可选择 PNG 或 JPEG 输出，并调整分辨率和质量以满足您的需求。它支持批量处理多个 PDF，并让您单独下载页面或打包为 ZIP。渲染通过 pdf.js 在您的浏览器中本地完成，因此您的文档会保留在您的设备上。

使用方法

  1. 添加一个或多个 PDF 文件进行转换。
  2. 选择 PNG 或 JPEG 输出，并设置 DPI（72–288），对于 JPEG，设置质量。
  3. 预览渲染后的页面以确认结果。
  4. 在本地将页面渲染为图像。
  5. 单独下载各个页面，或将所有内容一并打包为单个 ZIP。

常见问题

我的 PDF 在哪里处理？

使用 pdf.js 在您的浏览器中本地处理，因此文档绝不会被上传到服务器。

支持哪些输出格式和分辨率？

您可以将页面导出为 PNG 或 JPEG，并选择 72 到 288 之间的 DPI；JPEG 还提供可调的质量设置。

我可以一次性转换多个 PDF 吗？

可以。您可以批量处理多个 PDF 文件，并将所有生成的图像一起下载。

我如何下载提取的图像？

您可以一次下载一个页面，或将它们全部打包为单个 ZIP 文件。
