关于 JSON 格式化与校验器

JSON 格式化工具完全在你的浏览器中美化、压缩和校验 JSON——任何数据都不会离开你的设备。粘贴原始或杂乱的 JSON，即可瞬间将其美化为缩进一致的格式、折叠为单行，或在问题进入生产环境之前发现语法错误。

使用方法

  1. 将你的 JSON 粘贴或输入到左侧的输入编辑器中。
  2. 在配置面板中选择缩进风格（2 个空格、4 个空格或制表符）。
  3. 格式化并校验后的结果会立即显示在右侧——你可以复制、下载它，或以可折叠的树形结构查看。
  4. 如果 JSON 无效，编辑器会高亮显示错误，方便你在复制前修复它。

常见问题

我的 JSON 会被上传到服务器吗？

不会。所有格式化和校验都使用 JavaScript 在你的浏览器本地运行。你的数据永远不会离开你的设备，因此处理敏感载荷时也很安全。

美化和压缩有什么区别？

美化（格式化）会添加缩进和换行，使 JSON 易于人类阅读。压缩则会移除所有不必要的空白字符，生成最小的有效 JSON——这有助于减小 API 中的载荷大小。

它能处理大型 JSON 文件吗？

可以。由于处理是在客户端进行的，实际限制取决于你浏览器的内存。数兆字节的文件无需网络往返即可完成格式化。

为什么它提示我的 JSON 无效？

常见原因包括尾随逗号、使用单引号而非双引号、键未加引号或包含注释——严格的 JSON 不允许这些。错误信息会指出问题所在的位置。
