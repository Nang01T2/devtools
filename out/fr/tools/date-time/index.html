Description

Transformez des dates et horodatages entre divers formats, y compris les horodatages Unix, les chaînes ISO et les motifs personnalisés. Idéal pour le débogage et les intégrations API.

À propos de Convertisseur de format de date et heure

Les dates arrivent rarement dans le format dont vous avez besoin. Ce convertisseur traduit un même instant entre les horodatages Unix (en secondes ou millisecondes), les chaînes ISO 8601 et les formats lisibles, afin que vous puissiez lire les journaux, déboguer des API et réconcilier les différences de fuseaux horaires d'un coup d'œil. Tout est calculé entièrement dans votre navigateur, de sorte qu'aucune date ni horodatage ne quitte jamais votre appareil.

Comment utiliser

  1. Collez ou saisissez une valeur dans le champ d'entrée, comme un horodatage Unix ou une chaîne de date ISO 8601.
  2. Choisissez le format de votre entrée (l'outil peut aussi détecter automatiquement les formats courants).
  3. Choisissez le format cible et, le cas échéant, le fuseau horaire (UTC ou votre zone locale).
  4. Lisez le résultat converti et copiez-le pour l'utiliser dans votre code, vos tests ou vos requêtes API.

Questions fréquentes

Qu'est-ce qu'un horodatage Unix ?

Un horodatage Unix est le nombre de secondes écoulées depuis l'epoch Unix, soit 1970-01-01T00:00:00Z (minuit UTC le 1er janvier 1970). De nombreux systèmes utilisent aussi les millisecondes, qui sont 1000 fois plus grandes pour le même instant.

Quelle est la différence entre secondes et millisecondes ?

Les horodatages Unix en secondes comptent généralement 10 chiffres pour les dates actuelles, tandis que les horodatages en millisecondes en comptent 13. Date.now() de JavaScript renvoie des millisecondes, alors que de nombreux langages backend utilisent les secondes ; vérifiez donc l'ordre de grandeur avant de convertir.

Comment les fuseaux horaires sont-ils gérés ?

Les horodatages Unix et les chaînes ISO 8601 se terminant par Z sont absolus et liés à UTC. Lorsque vous affichez une date dans un format local lisible, le convertisseur applique un décalage, de sorte que le même instant peut afficher des heures d'horloge différentes selon les fuseaux horaires.

Mes données sont-elles envoyées à un serveur ?

Non. Toute l'analyse et la conversion se déroulent localement dans votre navigateur à l'aide des fonctions de date intégrées, de sorte que les dates et horodatages que vous saisissez ne sont jamais téléversés.
