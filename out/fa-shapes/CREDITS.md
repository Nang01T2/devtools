## What's vendored

Only a curated subset of Font Awesome Free's full ~2883-icon set is vendored
here (not the complete library) — see `manifest.json` for the exact list,
organized into 16 categories (14 mapping directly to Font Awesome's own
category taxonomy, one `brands` category sourced from Font Awesome's `brands`
style folder, and one hand-curated `other` miscellany bucket).

Every copied `.svg` file is byte-identical to its source in
`@fortawesome/fontawesome-free`, including its embedded Font Awesome license
comment — nothing has been modified.

Regenerated by `npm run build-fa-shapes` (`scripts/build-fa-shapes.mjs`); the
rest of this directory is gitignored and rebuilt from `node_modules` on
`postinstall`.
