Descripción

Codifica datos binarios o texto a Base64 para adjuntos de correo y API, o decodifícalos fácilmente con validación incorporada.

Acerca de Codificador y decodificador Base64 en línea

El Codificador / Decodificador Base64 convierte texto y datos binarios hacia y desde Base64 directamente en tu navegador. Base64 representa datos binarios como texto ASCII, lo que permite incrustarlos de forma segura en URL, JSON, XML, archivos adjuntos de correo y URI de datos. Codifica o decodifica al instante con validación integrada: nada se envía a un servidor.

Cómo usar

  1. Elige si tu origen es texto sin formato o un archivo.
  2. Para codificar, escribe o pega texto (o sube un archivo) en la entrada: la salida en Base64 aparece de inmediato.
  3. Para decodificar, pega una cadena Base64; el texto decodificado se muestra y se valida.
  4. Copia el resultado o descárgalo cuando trabajes con archivos.

Preguntas frecuentes

¿Es Base64 un cifrado?

No. Base64 es una codificación, no un cifrado. Cualquiera puede decodificarla: no ofrece seguridad ni confidencialidad. Úsala para el transporte seguro de datos binarios, no para proteger secretos.

¿Por qué la salida de Base64 es más larga que la entrada?

Base64 representa cada 3 bytes de entrada como 4 caracteres ASCII, por lo que la salida es aproximadamente un 33% más grande que los datos originales. Esa sobrecarga es el precio del transporte seguro como texto.

¿Qué es un URI de datos?

Un URI de datos incrusta un archivo directamente en una URL usando Base64, por ejemplo data:image/png;base64,iVBORw0K… Permite insertar imágenes o fuentes pequeñas en línea sin una solicitud HTTP aparte.

¿La decodificación ocurre en un servidor?

No. La codificación y la decodificación se ejecutan por completo en tu navegador, así que incluso los datos sensibles permanecen en tu dispositivo.
