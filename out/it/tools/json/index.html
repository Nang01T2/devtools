Informazioni su Formattatore e Validatore JSON

Il Formattatore JSON abbellisce, minimizza e convalida il JSON interamente nel tuo browser, senza che alcun dato lasci mai il tuo computer. Incolla JSON grezzo o disordinato per ottenere istantaneamente una formattazione leggibile con rientri coerenti, comprimerlo su una singola riga oppure individuare errori di sintassi prima che arrivino in produzione.

Come usare

  1. Incolla o digita il tuo JSON nell'editor di input a sinistra.
  2. Scegli uno stile di rientro (2 spazi, 4 spazi o tabulazione) dal pannello di configurazione.
  3. Il risultato formattato e convalidato appare istantaneamente a destra: copialo, scaricalo o visualizzalo come albero comprimibile.
  4. Se il JSON non è valido, l'editor evidenzia l'errore così puoi correggerlo prima di copiarlo.

Domande frequenti

Il mio JSON viene caricato su un server?

No. Tutta la formattazione e la convalida vengono eseguite localmente nel tuo browser usando JavaScript. I tuoi dati non lasciano mai il tuo dispositivo, il che lo rende sicuro per payload sensibili.

Qual è la differenza tra abbellire e minimizzare?

L'abbellimento (formattazione) aggiunge rientri e interruzioni di riga per rendere il JSON leggibile dall'uomo. La minimizzazione rimuove tutti gli spazi superflui per produrre il JSON valido più piccolo possibile, utile per ridurre la dimensione dei payload nelle API.

Può gestire file JSON di grandi dimensioni?

Sì. Poiché l'elaborazione avviene lato client, il limite pratico è la memoria del tuo browser. File di diversi megabyte vengono formattati senza alcuna richiesta di rete.

Perché dice che il mio JSON non è valido?

Le cause comuni sono virgole finali, apici singoli invece di doppi, chiavi senza virgolette o commenti, nessuno dei quali è consentito nel JSON in senso stretto. Il messaggio di errore indica la posizione del problema.
