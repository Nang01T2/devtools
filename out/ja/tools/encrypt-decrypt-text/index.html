概要： テキスト暗号化・復号ツール

このツールは、あなたが提供する鍵を使って、AES などのアルゴリズムでテキストの本物の対称暗号化と復号を行います。対称とは、暗号化と復号の両方に同じ秘密鍵を使うことを意味するため、暗号文を共有する相手にも別の安全な経路で鍵を渡す必要があります。秘密なしで誰でも元に戻せて機密性を提供しない Base64 や URL エンコードとは異なり、これは本物の暗号化です。正しい鍵がなければ平文を復元できません。すべての処理はブラウザ内でクライアントサイドで実行されます。

使い方

  1. 暗号化したいか復号したいかを選択します。
  2. テキスト（暗号化する平文、または復号する暗号文）と秘密鍵を入力します。
  3. オプションがあれば、アルゴリズムとモードを選択します。
  4. 操作を実行し、生成された暗号文または復元された平文をコピーします。
  5. 鍵は暗号文と一緒にではなく、安全な経路で受信者と共有してください。

よくある質問

エンコードは暗号化と同じですか？

いいえ。エンコード（Base64 や URL エンコードなど）は誰でも元に戻せて、機密性を提供しません。暗号化には秘密鍵が必要で、その鍵がなければ元のテキストを復元できません。これが機密性を与えるものです。

ここでの対称暗号化とは何を意味しますか？

同じ鍵でテキストを暗号化し復号することを意味します。双方が同一の秘密鍵を持つ必要があるため、鍵は安全に交換し、秘密に保たなければなりません。

私のテキストや鍵はどこかにアップロードされますか？

いいえ。暗号化と復号は完全にブラウザ内で行われるため、平文、暗号文、鍵はあなたのデバイスに留まります。

鍵を失ったり打ち間違えたりするとどうなりますか？

正しい対称暗号化では、正確な鍵がなければ暗号文を復号できません。鍵を失ったり間違えたりすると、元のテキストは事実上復元不可能になります。
