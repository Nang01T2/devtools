将 SVG 转换为 PNG

当某个平台要求像素图像时 — 文档、电商平台、应用商店或设计交付 — 将 SVG 矢量栅格化为 PNG。PNG 会在你导出的分辨率下保留 SVG 的透明度和锐利边缘。

打开 SVG → PNG 转换器

免费 · 无需注册 · 无需上传 — 100% 在你的浏览器中运行

如何将 SVG 转换为 PNG

  1. 打开转换器并添加你的 SVG 文件 — 拖放、从剪贴板粘贴，或选择整个文件夹。
  2. 在选项面板中将 PNG 选为输出格式，然后点击应用。
  3. 如有需要，可微调质量或调整尺寸 — 文件会即时重新编码。
  4. 可单独下载结果、打包为 ZIP，或合并成一个 PDF。

关于这些格式

SVG

SVG 是一种矢量格式 — 它用数学方式描述图形而非存储像素，因此可以缩放到任意尺寸而不损失质量。许多工具、文档和平台只接受栅格图像，这时你就需要按选定的分辨率将 SVG 栅格化。

PNG

PNG 是一种支持完整透明度的无损格式，非常适合截图、界面图形、徽标，以及任何带有锐利边缘或文字的图像。代价是体积：以 PNG 存储的照片内容比 JPEG、WebP 或 AVIF 大得多。

常见问题

我的 SVG 文件会被上传到服务器吗？

不会。转换完全在你的浏览器中通过 WebAssembly 编解码器运行 — 你的文件永远不会离开你的设备。首次访问之后，转换器甚至可以离线工作。

我可以一次转换多个文件吗？

可以 — 想添加多少文件（或整个文件夹）都行，一批转换完成后，把所有结果打包成一个 ZIP 下载。没有文件数量或大小限制。

将 SVG 转换为 PNG 会损失质量吗？

你始终掌控全局：PNG 的质量可在选项面板中调节，默认值经过调校，对于常见图片肉眼几乎无法分辨。你可以随时用不同设置重新转换。

我的 PNG 会是什么分辨率？

默认使用 SVG 的固有尺寸，你也可以在转换前用调整尺寸选项设定精确的输出尺寸（例如适配宽度 1024）。
