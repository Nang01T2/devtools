关于 RSA 密钥对生成器

本工具生成 RSA 密钥对，这是一种非对称密码学原语，由一个公钥和一个与之匹配的私钥组成。公钥可以自由共享，用于加密数据或验证签名；而私钥必须保密，用于解密数据或创建签名。您可以选择密钥长度，当今 2048 位被视为实际最低标准，4096 位则提供更大的安全余量。密钥在您的浏览器客户端生成，绝不会被传输。

使用方法

  1. 选择密钥长度，例如 2048 或 4096 位。
  2. 点击「生成」，在您的浏览器中创建一个全新的公钥/私钥对。
  3. 复制或下载公钥以与他人共享。
  4. 妥善存储私钥，切勿与任何人共享。

常见问题

公钥和私钥有什么区别？

RSA 是非对称的。公钥用于加密数据和验证签名，可以公开共享；私钥用于解密数据和创建签名，必须保密。

我应该选择什么密钥长度？

2048 位通常被认为是当今可接受的最小尺寸，4096 位以较慢的运算为代价提供额外的长期安全性。

私钥会被发送到服务器吗？

不会。密钥对完全在您的浏览器中生成，因此除非您明确保存或共享，否则私钥将保留在您的设备上。

我可以将 RSA 同时用于加密和签名吗？

可以。RSA 支持加密（用公钥加密，用私钥解密）和数字签名（用私钥签名，用公钥验证）。
