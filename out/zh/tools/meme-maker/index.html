在浏览器中制作表情包——为任何照片添加文字和贴纸，包括 iPhone 的 HEIC 图片。完全私密，无需上传，无水印，无需注册。

关于 表情包制作工具

直接在浏览器中制作表情包——以照片、动态 GIF 或 MP4/WebM 视频为底图，包括 iPhone 拍摄的 HEIC 照片。对每一帧应用 60 多种滤镜与特效，添加文字、贴纸和叠加图层，然后导出为 PNG、JPEG、WebP、AVIF、GIF 或 MP4。从 86 个内置模板（81 个图片模板和 5 个动态 GIF 模板）中选择，或从自己的文件开始。所有操作均在设备本地完成：无需上传、无水印、无需账号。

使用方法

  1. 将照片、动态 GIF 或 MP4 视频拖放到画布——或从剪贴板粘贴，或从 86 个内置模板（图片或动态 GIF）中选择。
  2. 点击“添加文字”放置标注文字，在画布上拖动并调整大小，然后在工具栏中调整字体、填充颜色和描边。
  3. 可选：使用“添加图片”在上方叠加贴纸图片。
  4. 点击属性面板中的“图片”标签，应用滤镜（25 种预设，如 Oceanic、Vintage、Radio）或一键特效（60 多种，包括 Sepia、Invert、Solarize、Halftone、Oil Paint）。对于动态 GIF 或视频底图，滤镜会应用到所有帧。
  5. 点击画布上的播放按钮，预览带有叠加图层的动态 GIF 或视频。点击“恢复原始”可撤销所有滤镜并还原原始动画。
  6. 在图层面板中用上下箭头调整图层顺序，或点击垃圾桶图标删除图层。
  7. 选择导出格式并点击下载。照片选 PNG/JPEG/WebP/AVIF，保留动画选 GIF，需要保留原视频音频则选视频 MP4（浏览器支持 WebCodecs 时导出 MP4，否则回退为 WebM）。

常见问题

真的免费且没有水印吗？

是的。该工具完全在浏览器中运行，不经过任何服务器。没有水印、不需要账号、完全免费。

可以用动态 GIF 或视频作为底图吗？

可以。拖入任意 GIF 文件，它会作为表情包底图播放——你可以添加文字标注、对每一帧应用滤镜，并重新导出为 GIF。MP4 和 WebM 视频同样支持：导出为视频 MP4 时音频会被完整保留。

滤镜可以应用到动态 GIF 和视频吗？

可以。在动态 GIF 底图上点击滤镜或特效时，会逐帧应用到整个动画。筛选后 GIF 导出选项仍然可用，随时可以点击“恢复原始”还原未经滤镜处理的原始状态。

可以用 iPhone 的 HEIC 照片制作表情包吗？

可以。HEIC 文件直接在浏览器中解码，无需任何转换步骤。大多数在线表情包工具根本无法处理 HEIC。

我的照片会被上传吗？

不会。所有操作——解码、合成、滤镜处理和导出——均在您的设备上本地运行，没有任何数据发送到服务器。

可以导出哪些图片格式？

PNG（无损，最适合文字较多的表情包）、JPEG（文件更小，适合照片）、WebP 和 AVIF（现代格式，压缩效果出色）、GIF（保留带文字标注的动画）以及视频 MP4（导出带原视频音频的动画——浏览器支持 WebCodecs 时为 MP4，否则回退为 WebM）。所有静态导出均以 2× 像素比输出，在高分辨率屏幕上显示清晰锐利。

内置模板可以商业使用吗？

古典艺术模板属于公共领域（艺术家均于 1928 年前去世）。版式模板和动态 GIF 模板为原创作品，无版权限制。三张图库照片采用 CC BY 或 CC BY-SA 授权——注明出处即可商业使用。完整署名信息请参阅 public/meme-templates/CREDITS.md。
