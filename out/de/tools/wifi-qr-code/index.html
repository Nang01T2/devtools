Über WLAN-QR-Code-Generator

Dieser Generator verwandelt Ihre Netzwerk-Anmeldedaten in einen standardmäßigen WIFI:-QR-Code, sodass Gäste Ihrem WLAN durch Scannen statt durch Eingabe eines langen Passworts beitreten können. Der QR kodiert SSID, Sicherheitstyp und Passwort direkt im Bild im Format WIFI:T:WPA;S:<ssid>;P:<password>;;, das iOS- und Android-Kameras nativ erkennen. Alles wird in Ihrem Browser erstellt, sodass Ihr Netzwerkname und Ihr Passwort niemals Ihr Gerät verlassen oder einen Server berühren.

Verwendung

  1. Geben Sie Ihren Netzwerknamen (SSID) genau so ein, wie er erscheint, einschließlich korrekter Groß- und Kleinschreibung.
  2. Wählen Sie den Sicherheitstyp — WPA/WPA2, WEP oder kein Passwort (offenes Netzwerk).
  3. Geben Sie das Passwort ein (überspringen Sie dies bei einem offenen Netzwerk) und aktivieren Sie 'versteckte SSID', falls Ihr Netzwerk seinen Namen nicht ausstrahlt.
  4. Passen Sie optional die Farben an und sehen Sie sich dann eine Vorschau des erzeugten QR-Codes an.
  5. Laden Sie das PNG herunter und drucken Sie es oder teilen Sie es dort, wo Gäste es scannen können.

Häufige Fragen

Was wird tatsächlich im WLAN-QR-Code gespeichert?

Der Code kodiert eine Klartext-Zeichenkette WIFI:T:WPA;S:<ssid>;P:<password>;;. SSID, Sicherheitstyp und Passwort sind alle direkt im Bild eingebettet, weshalb der QR statisch ist und offline ohne jede Abfrage funktioniert.

Wird mein WLAN-Passwort irgendwohin gesendet, wenn ich den Code erzeuge?

Nein. Der QR-Code wird vollständig clientseitig in Ihrem Browser erzeugt. Ihre SSID und Ihr Passwort werden niemals auf einen Server hochgeladen, sodass das Passwort nur in dem Bild existiert, das Sie erstellen und zum Teilen auswählen.

Welche Sicherheitstypen werden unterstützt?

WPA und WPA2, WEP sowie offene Netzwerke ohne Passwort. Sie können auch eine versteckte SSID kennzeichnen, damit Geräte beim Scannen wissen, dass sie nach einem nicht ausstrahlenden Netzwerk suchen müssen.

Funktioniert der QR-Code nicht mehr, wenn ich die Netzwerkeinstellungen unverändert lasse?

Nein. Die Anmeldedaten sind in das Bild eingebrannt, sodass der QR weiterhin funktioniert, solange SSID, Sicherheitstyp und Passwort gleich bleiben. Wenn Sie das Passwort ändern, müssen Sie einen neuen Code erzeugen.
