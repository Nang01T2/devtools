Descripción

Analiza estadísticas de texto como el conteo de palabras y la frecuencia de caracteres, mientras conviertes entre camelCase, snake_case, PascalCase y más para un código limpio.

Acerca de Inspector de texto y conversor de mayúsculas y minúsculas

Esta herramienta inspecciona cualquier texto que pegues y lo desglosa en estadísticas útiles como el número de palabras, caracteres y líneas, y luego te permite convertir el mismo texto entre estilos de nomenclatura de programación comunes. Todo se ejecuta completamente en tu navegador, por lo que tu texto nunca sale de tu dispositivo. Úsala para limpiar identificadores, normalizar nombres de variables o auditar rápidamente el tamaño y la estructura de un fragmento.

Cómo usar

  1. Pega o escribe tu texto en el área de entrada.
  2. Lee las estadísticas en vivo, incluido el número de palabras, caracteres y líneas.
  3. Elige un estilo de destino como camelCase, snake_case, PascalCase, kebab-case, MAYÚSCULAS, minúsculas o Título.
  4. Copia la salida convertida para usarla en tu código o documentos.

Preguntas frecuentes

¿Qué formatos de nomenclatura son compatibles?

Puedes convertir texto a camelCase, snake_case, PascalCase, kebab-case, MAYÚSCULAS, minúsculas y Título.

¿Se envía mi texto a un servidor?

No. Toda la inspección y conversión ocurren localmente en tu navegador, por lo que nada se sube ni se almacena de forma remota.

¿Qué estadísticas muestra el inspector?

Informa de recuentos como palabras, caracteres y líneas para que puedas evaluar rápidamente el tamaño y la estructura de tu texto.

¿Puedo convertir una frase de varias palabras en un solo identificador?

Sí. Las frases de varias palabras se unen según el estilo elegido; por ejemplo, camelCase produce un único token con capitalización interna.
